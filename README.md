# Portfólio Premium - Maquiadora Profissional

Este é um projeto de landing page/portfólio desenvolvido para profissionais de maquiagem que buscam transmitir sofisticação, confiança e alto padrão em seus serviços (Noivas, Formaturas, Festas de Gala e Eventos Casuais).

## 🌟 Diferenciais do Projeto

- **Design Premium & Minimalista:** Interface moderna com uso estratégico de tipografia elegante, espaços em branco e um contraste refinado.
- **Micro-interações e Animações Suaves:** Efeitos de hover magnéticos nos botões, zoom suave interativo nas fotos de destaque (como no rodapé) e fade-ins ao rolar a página (scroll reveal) para uma experiência de alta performance.
- **Portfólio Interativo (Bento Grid):** Os cards de serviços funcionam como capas dinâmicas que alternam fotos automaticamente (slideshow) a cada 4 segundos, dando vida à página.
- **Galeria Lightbox Exclusiva:** Ao clicar em um serviço, o usuário é imerso em uma galeria de tela cheia (com navegação por setas) contendo uma curadoria específica para aquele nicho, sem poluir a interface principal.
- **Foco em Conversão (CTAs Estratégicos):** Botões de agendamento estrategicamente posicionados e integrados com a API do WhatsApp para facilitar o contato direto.
- **Totalmente Responsivo:** Experiência de uso perfeita tanto em desktops quanto em dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação semântica.
- **CSS3 (Vanilla):** Variáveis nativas, CSS Grid, Flexbox, Glassmorphism, animações avançadas. Nenhum framework pesado, garantindo carregamento ultrarrápido.
- **JavaScript (Vanilla):** Lógica para o preloader, scroll reveal automático, carrossel de imagens dinâmico nos cards e manipulação do DOM para a Galeria Lightbox.
- **Phosphor Icons:** Biblioteca de ícones moderna e leve.

## 📁 Estrutura de Arquivos

- `index.html`: Estrutura principal da landing page.
- `style.css`: Estilização completa do projeto, variáveis de cor, responsividade e animações.
- `script.js`: Lógica de interatividade (Galeria, Slideshow, Preloader).
- Pastas de Mídia (`Noivas/`, `Formatura/`, `Festa de gala/`, `eventos/`): Imagens organizadas por categoria para as galerias.

## 🚀 Como Usar e Customizar

1. **Configurar o WhatsApp:** No arquivo `index.html`, procure por `55SEUNUMEROAQUI` e substitua pelo número real da profissional (incluindo código do país e DDD).
2. **Redes Sociais:** No rodapé do arquivo `index.html`, substitua os links do Instagram e Facebook trocando `SEU_USUARIO_AQUI` pelos perfis oficiais.
3. **Imagens:** Substitua `Logo.png`, `Livia Duarte.jpg` e as fotos dentro das pastas pelos assets reais da profissional mantendo o alto padrão visual.
